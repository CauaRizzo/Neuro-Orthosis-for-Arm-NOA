# Neuro Orthosis for Arm (NOA)

Projeto acadêmico desenvolvido na disciplina de Projeto Integrador.
O objetivo do projeto é o desenvolvimento de uma órtese mioelétrica vestível
de baixo custo, baseada na aquisição e processamento de sinais EMG.

## Estrutura do Repositório

- Hardware: esquemático e layout da PCB (Altium Designer)
- Firmware: código embarcado para STM32
- Documentation: relatório técnico e imagens

## Observações

Este repositório possui finalidade educacional.
A definição de uma licença open-source será realizada em versões futuras.
